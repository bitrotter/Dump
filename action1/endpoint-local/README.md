# Action1 Endpoint Scripts

Scripts that run locally on each endpoint, deployed via Action1 "Run Script" feature.

These scripts check or correct issues on individual machines.

## Categories

- `disk/` - Disk space and health checks
- `windows-update/` - Windows Update status and fixes
- `services/` - Service health and management
- `event-logs/` - Event log analysis
- `maintenance/` - General maintenance tasks

## Usage

Deploy via Action1 Console > Run Script on selected endpoints or groups.
